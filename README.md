# Social-Sphere
Social Sphere is a console-based social networking application built in C++, designed to demonstrate the principles of Object-Oriented Programming (OOP).

Features:
Add User: Create and manage user profiles.
Add Post: Share updates and content with others.
Add Comment: Engage with posts through comments.
Add Like: Show appreciation by liking posts.
This project showcases the power and flexibility of OOP in C++, providing a modular and scalable codebase for social networking functionalities.
